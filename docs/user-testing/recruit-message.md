# Mensagem de Recrutamento

> Copie e adapte para Slack, WhatsApp, LinkedIn ou email.
> Envie para devs que usam Claude Code, Cursor ou similar no dia a dia.

---

## Versão curta (Slack / WhatsApp)

```
Ei! Estou testando uma ferramenta que organiza como você trabalha com IA no desenvolvimento.
Leva ~20 minutos e você usa no seu próprio projeto.

Sem nada para instalar antes — só precisar ter Claude Code (ou Cursor).

Topa? Te mando o guia.
```

---

## Versão longa (email / LinkedIn)

```
Assunto: 20 minutos para testar uma ferramenta de IA — preciso de feedback real

Olá!

Estou desenvolvendo o Synapos — um workflow system que organiza como você usa IA 
em projetos reais. Não é mais um "agent mágico" — é uma estrutura que faz a IA 
trabalhar melhor no contexto do seu projeto específico.

Procuro 5 devs para um teste de uso real:
  • Você usa no seu próprio projeto (existente ou novo)
  • ~20 minutos no total, divididos em 2 sessões curtas
  • Feedback honesto — especialmente onde travar ou não entender

Requisito: usar Claude Code ou Cursor no dia a dia.

Topa? Responde aqui e te mando o guia de instalação.
```

---

## Perfil ideal dos testadores

Priorize nesta ordem:

1. Dev que usa Claude Code ou Cursor ativamente (pelo menos 3x/semana)
2. Tem projeto real em andamento (não projeto de estudo)
3. Nunca ouviu falar do Synapos antes
4. Não é amigo próximo — evita viés de querer agradar

---

## O que NÃO fazer ao recrutar

- Não explique como o sistema funciona antes do teste
- Não diga "é simples" ou "é fácil" — isso cria expectativa
- Não escolha só pessoas que vão dar feedback positivo
- Não recrute mais de 1 pessoa do mesmo contexto (ex: dois devs do mesmo time)
