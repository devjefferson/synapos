# Aquecimento do Projeto

Para se preparar para esta sessão, por favor:


1 - Master Docs

- Revise o index.md dos MASTER DOCS deste projeto e memorize-o para que você saiba quais arquivos pode ler mais tarde quando a necessidade surgir.
- Os master docs estão nesta pasta: ~/{project-name}


2 - Revise o README.md da raiz deste projeto. Além disso, mantenha uma lista dos arquivos na pasta docs/ em seu contexto para que você possa se referir a eles mais tarde quando a necessidade surgir

Você pode acessar estes arquivos diretamente através do plataforma de arquivos ou através da ferramenta code-expert que você tem acesso.

O nome do projeto será fornecido como argumentos: #$ARGUMENTS
