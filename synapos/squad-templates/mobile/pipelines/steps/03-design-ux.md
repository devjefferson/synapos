# Design UX Mobile

**Agent:** Viviane Visual 🎨

## Tarefa

1. Revise o fluxo de usuário para a feature
2. Verifique zonas de toque e tamanhos mínimos (44pt iOS / 48dp Android)
3. Identifique padrões de plataforma a seguir (iOS vs Android)
4. Revise acessibilidade: labels, contraste, font scaling
5. Defina estados visuais: loading, error, empty, success

## Output

Salve em `docs/ux-notes.md` com:
- Notas de UX por tela
- Padrões de plataforma aplicados
- Checklist de acessibilidade
- Decisões de design
